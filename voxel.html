voxel — 2009

graphic editor — programming

The main purpose of the project was to design a graphic editor with voxels, a tool for creating 3D models. The user can interact with the application and control the pointer, managing the cubes and drawing a lot of different models. Some of the editor features are transparencies, illumination and textures. It’s also possible to save the draw.
