maria — 2012

editorial design — magazine covers

Maria is a popular magazine in Portugal. The main goal of the project was to create book covers with Markov Chains. All the text is created using Markov Chains and the image of the book cover is generated by the main sentence (e.g. Bieber colecciona mansões). This way, the words are surveyed in a search engine (e.g. bing) and the first image is added to the book cover.
